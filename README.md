# 🤖 Agente de Atendimento de IA no WhatsApp

**`n8n | WhatsApp | IA | Automação de Atendimento`**

Projeto de automação desenvolvido com foco em **atendimento inteligente via WhatsApp**, utilizando **n8n + IA** para otimizar o primeiro contato com clientes, qualificar leads e registrar informações automaticamente.

---

## 🎯 Objetivo

Criar um fluxo automatizado de atendimento via WhatsApp com foco em:

* respostas automáticas
* qualificação inicial de leads
* direcionamento para atendimento humano
* registro estruturado das informações

---

## 🛠 Tecnologias

* **n8n**
* **WhatsApp API**
* **Webhooks**
* **IA (ChatGPT / Claude)**
* **Google Sheets / Supabase**

---

## 🔄 Fluxo da automação

```text
Cliente envia mensagem
        ↓
Webhook recebe
        ↓
n8n processa
        ↓
IA interpreta a intenção
        ↓
Resposta automática
        ↓
Registro do lead
```
